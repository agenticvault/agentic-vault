## Background

發佈 `@agenticvault/agentic-vault` v0.1.0，包含完整的 MCP server、CLI（8 subcommands）、Claude Code plugin、DeFi protocol integration（ERC-20 + Uniswap V3 + Aave V3）、PolicyEngine V2、OpenClaw plugin。

> **Note**: 套件名稱已從 `@sd0xdev/agentic-vault` 遷移至 `@agenticvault/agentic-vault`，詳見 [namespace-migration](./2026-02-15-namespace-migration.md)。
